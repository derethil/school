HTML Elements Cheatsheet

<html>
  • purpose: the top level element of an HTML document; must be unique and all elements must be descendents
  • display: neither
  • parents: none
  • contents:
    1. <head>
    2. <body>
<head>
  • purpose: container for metadata about the HTML document; not visible
  • display: neither
  • parents: <html>
  • contents:
    1. <title>
    2. <link>
<title>
  • purpose: HTML document title
  • display: neither
  • parents: <head>
  • contents: text content
<meta>
  • purpose: define HTML document metadata
  • display: N/A
  • parents: <head>
  • contents: none; data is enemd using attributes
  • attributes:
    • <charset>: specifies the charcter encoding for the HTML document
<link>
  • purpose: defines the relationship between the current document and an external resource
  • display: neither
  • parents: <head>
  • contents: N/A
<body>
  • purpose: defines the document's body; contains the visible document
  • display: block-level
  • parents: <html>
  • contents:
    1. <div>
    2. <ul>
    3. <ol>
<div>
  • purpose: defines the document's body; contains the visible document
  • display: block-level
  • parents: <html>
  • contents: any
<h1>
  • purpose: define the most important heading in the document
  • display: block-level
  • parents: any
  • contents: flow content
<ul>
  • purpose: an unordered list; uses bullet points by default
  • display: block-level
  • parents: any
  • contents: <li>
<ol>
  • purpose: an ordered list; uses numbers by default
  • display: block-level
  • parents: any
  • contents: <li>
<li>
  • purpose: list element in an ordered or unordered list
  • display: block-level
  • parents:
    1. <ul>
    2. <ol>
  • contents: flow content
<em>
  • purpose: define a section of text as formatted em
  • display: in-line
  • parents: any
  • contents: text content

CSS Selectors Cheatsheet

Element Selector

  • purpose: selects all elements that are of type element
  • example: h3 { font-style: italic; }
  • used in combination with :nth-of-type(odd) to create zebra stripes as well as with :nth-child(1) to select the first element

Class Selector [.]

  • purpose: selects all elements with given class
  • example: .post { padding: 1em; }
  • to give an element a class, give it an attribute class like so: <p class="content">

Id Selector [#]

  • purpose: selects the element with given id
  • example: #nav { background-color: lightsteelblue; }
  • to give an element an id, give it an attribute id like so: <p id="content">

CSS Properties Cheatsheet

background-color

  • purpose: change the background color of selected elements
  • values: hex, rgb, rgba, hsl, hsla, predefined colors

color

  • purpose: change the text color of selected elements
  • values: hex, rgb, rgba, hsl, hsla, predefined colors

font-family

  • purpose: change the font of selected elements
  • values: predefined fonts, imported fonts

font-size

  • purpose: change the text size of selected elements
  • values: pixels, em, rem, percent, etc, plus predefined values such as medium, smaller, larger

margin

  • purpose: gives margin to selected elements
  • values: pixels, em, rem, percent, etc

padding

  • purpose: gives padding to selected elements
  • values: pixels, em, rem, percent, etc

list-style-type

  • purpose: change the style of a list
  • values: circle, square, upper-roman, lower-alpha, etc

border-radius

  • purpose: defines the radius of the selected element's corners
  • values: pixels

width

  • purpose: specifies the width of the selected element
  • values: pixels, em, rem, percent, etc

height

  • purpose: change the height of the selected element
  • values: pixels, em, rem, percent, etc
